## :cloud: Installation

```sh
git clone https://github.com/ayatmaulana/ay-md-progress
```

## :rocket: Run 

#### Manual

```bash
cd ay-md-progress

#if you using yarn
yarn

#if you using npm
npm install
```

#### Docker

```bash
docker-compose up -d
```
